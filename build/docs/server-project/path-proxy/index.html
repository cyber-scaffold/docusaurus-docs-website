正式环境接口代理转发

为什么这些接口要做转发#

有些接口的地址在appapi.yi-you.org上此时,此时如果在yi-you.org中访问appapi.yi-you.org的接口的话就会出现跨域问题

白板的服务端渲染接口#

源服务接口地址

http://appapi.yi-you.org/pano_mobile/**/*

代理服务地址

http://yi-you.org/pano_mobile/**/*
